コンパイル方法（TexShopの場合）：
1. "syspro_pp_xxxxxxxx_name.tex" を開く．
2. "syspro_pp_xxxxxxxx_name.tex" を LaTex で1回コンパイルする．
3. "syspro_pp_xxxxxxxx_name.tex" を BibTex で1回コンパイルする．エラーが出ないことを確認する．
4. "syspro_pp_xxxxxxxx_name.tex" を LaTex で2, 3回コンパイルする．
5. "syspro_pp_xxxxxxxx_name.pdf" が生成される．最終ページに "参考文献" が生成されていることを確認する．

.bibファイルについて：
.bib ファイルを使うと，tex で参考文献リストを簡単に管理できます．
下記ウェブサイトなどを参考に，ぜひ使い方を勉強してください．
